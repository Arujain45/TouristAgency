Краткая инструкция по установке программы Туристическое Агентство.
Инструкция для версии программы 0.4 (прототип).
Программа написана в Embarcadero C++ Builder XE.

1).  Создайте на диске C:\ каталог Software.
2).  Переместите файл с базой данных AgencyBase.mdb из папки Database каталога программы в папку C:\Software.
3).  Запустите скрипт Check ODBC Base.vbs из папки Database\Scripts, для проверки на существование пользовательского DSN. Если будет сообщение о существовании DSN: "Пользовательский DSN 'TouristAgency Base' существует.", то на этом настройка закончена. В противном случае переходим к шагу 3*.
3*). Добавьте в DSN с именем "TouristAgency Base", драйвером для работу с *.mdb, указав путь к базе данных (по умолчанию C:\Software\AgencyBase.mdb). Для автоматизации этого процесса можно воспользоваться *.reg файлом ODBC TouristAgency Base.reg из папки Database\Scripts.
4).  Запуск программы осуществляется через выполнение файла TouristAgency.exe из папки Release.

P.S.: При удалении программы не забудьте удалить пользовательский DSN.
